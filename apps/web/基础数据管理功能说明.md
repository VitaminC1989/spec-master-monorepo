# 📊 基础数据管理功能说明

## ✅ 已完成的功能

新增三个基础数据管理模块，每个模块都支持完整的增删改查（CRUD）操作：

2. **📏 尺码管理** - 管理尺码规格

## 📝 使用方法

### 客户管理

#### 新建客户
1. 点击右上角"新建客户"按钮
2. 填写表单：
   - 客户名称（必填）
   - 联系人、电话、邮箱（可选）
   - 地址、备注（可选）
3. 点击"创建"按钮
4. 新客户出现在列表中

#### 编辑客户
1. 点击客户行的"编辑"按钮
2. 修改客户信息
3. 点击"确定"保存

#### 删除客户
1. 点击客户行的"删除"按钮
2. 在确认对话框中点击"确认删除"
3. 客户从列表中删除

---

### 尺码管理

#### 新建尺码
1. 点击"新建尺码"按钮
2. 填写表单：
   - 尺码代码（如：XXL）
   - 尺码名称（如：特大号）
   - 排序序号（如：7）
   - 是否启用（默认启用）
3. 点击"创建"
4. 新尺码出现在列表中

#### 排序功能
- 排序序号越小，显示越靠前
- 可以通过编辑修改排序
- 列表会按排序序号自动排列

#### 启用/禁用
- 启用：显示绿色"启用"标签
- 禁用：显示红色"禁用"标签
- 通过编辑可以切换状态

---

### 单位管理

#### 新建单位
1. 点击"新建单位"按钮
2. 填写表单：
   - 单位代码（如：yard）
   - 单位名称（如：码）
   - 单位类型（如：长度）
   - 是否启用（默认启用）
3. 点击"创建"

#### 按类型分类
- 长度类：米、厘米、毫米等
- 数量类：条、粒、套等
- 重量类：千克、克等

## 🔧 技术实现

### 数据模型（TypeScript）

```typescript
// 客户
interface ICustomer {
  id: number;
  customer_name: string;
  contact_person?: string;
  contact_phone?: string;
  contact_email?: string;
  address?: string;
  note?: string;
  create_date: string;
}

// 尺码
interface ISize {
  id: number;
  size_code: string;
  size_name: string;
  sort_order?: number;
  note?: string;
  is_active: boolean;
}

// 单位
interface IUnit {
  id: number;
  unit_code: string;
  unit_name: string;
  unit_type?: string;
  note?: string;
  is_active: boolean;
}
```
