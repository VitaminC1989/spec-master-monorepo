# 🚀 SpecMaster 部署指南

## 📦 构建产物说明

构建后的 `dist/` 目录包含：
- `index.html` - 入口 HTML 文件
- `assets/` - CSS 和 JS 文件

**重要：** 这是一个 SPA（单页应用），**不能直接双击打开 index.html**，需要通过 HTTP 服务器访问。

---

## 🌟 给甲方看的最佳方案

### ✅ 方案1：Vercel 部署（最推荐）

**优点：**
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 全球 CDN 加速
- ✅ 甲方只需要一个链接即可访问
- ✅ 支持自定义域名

**部署步骤：**

#### 方法A：使用 Vercel CLI（推荐）

```bash
# 1. 安装 Vercel CLI（全局安装一次即可）
npm install -g vercel

# 2. 登录 Vercel（首次使用需要）
vercel login

# 3. 部署项目
cd /Users/yqchen/Desktop/Projects/SpecMaster
vercel

# 4. 按提示操作：
#    - Set up and deploy? → Yes
#    - Which scope? → 选择你的账号
#    - Link to existing project? → No
#    - Project name? → specmaster-demo（或自定义）
#    - Directory? → ./（回车，使用当前目录）
#    - Override build command? → No
#    - Override output directory? → dist

# 5. 部署完成后会给你一个链接，如：
#    https://specmaster-demo.vercel.app
```

#### 方法B：通过 Vercel 网站部署

1. 访问 https://vercel.com/
2. 使用 GitHub、GitLab 或 Bitbucket 登录
3. 点击 "Add New..." → "Project"
4. 导入你的 Git 仓库（如果没有，先推送到 GitHub）
5. Vercel 自动检测配置，点击 "Deploy"
6. 等待部署完成，获得访问链接

**结果：**
```
✅ 部署成功！
访问链接：https://specmaster-demo-xxx.vercel.app

分享给甲方：
直接把链接发给甲方，甲方在浏览器中打开即可使用
无需安装任何软件！
```

### ✅ 方案2：Netlify 部署（同样推荐）

**优点：**
- ✅ 完全免费
- ✅ 操作简单
- ✅ 自动 HTTPS
- ✅ 支持拖拽上传

**部署步骤：**

#### 方法A：拖拽上传（最简单）

1. 访问 https://app.netlify.com/drop
2. 将构建后的 `dist` 文件夹拖拽到页面上
3. 等待上传完成（几秒钟）
4. 获得访问链接，如：`https://random-name-123.netlify.app`

#### 方法B：使用 Netlify CLI

```bash
# 1. 安装 Netlify CLI
npm install -g netlify-cli

# 2. 登录 Netlify
netlify login

# 3. 部署
cd /Users/yqchen/Desktop/Projects/SpecMaster
netlify deploy

# 4. 按提示操作：
#    - Create & configure a new site? → Yes
#    - Team? → 选择你的团队
#    - Site name? → specmaster-demo（可选）
#    - Publish directory? → dist

# 5. 预览部署（草稿链接）
#    会给你一个预览链接

# 6. 正式部署（生产环境）
netlify deploy --prod
```

**结果：**
```
✅ 部署成功！
访问链接：https://specmaster-demo.netlify.app
```

### ✅ 方案3：使用静态文件服务（本地演示）

如果甲方在您旁边，可以快速启动一个本地服务器：

#### 方法A：使用 serve（推荐）

```bash
# 1. 全局安装 serve（只需一次）
npm install -g serve

# 2. 启动服务
cd /Users/yqchen/Desktop/Projects/SpecMaster
serve dist

# 3. 访问显示的链接，如：
#    http://localhost:3000
```

#### 方法B：使用 http-server

```bash
# 1. 全局安装 http-server
npm install -g http-server

# 2. 启动服务
cd /Users/yqchen/Desktop/Projects/SpecMaster
http-server dist -p 8080

# 3. 访问：
#    http://localhost:8080
```

#### 方法C：使用 Python（无需安装）

```bash
# Python 3
cd /Users/yqchen/Desktop/Projects/SpecMaster/dist
python3 -m http.server 8000

# 访问：http://localhost:8000
```

**注意：** 这种方式需要甲方和您在同一网络，或者您需要向外网暴露端口。

## 🔧 高级配置

### 自定义域名（可选）

如果您有自己的域名：

**Vercel：**
1. 在 Vercel 项目设置中点击 "Domains"
2. 添加你的域名（如 demo.yourcompany.com）
3. 按提示配置 DNS 记录

**Netlify：**
1. 在 Netlify 项目设置中点击 "Domain management"
2. 添加自定义域名
3. 配置 DNS 记录

## 🐛 常见问题

### Q: 部署后刷新页面 404？
A: 需要配置路由重定向，`vercel.json` 已配置好

### Q: 图片不显示？
A: 检查图片路径是否正确，使用相对路径

### Q: 数据会丢失吗？
A: Demo 使用 Mock 数据，刷新会重置，这是正常的

### Q: 可以免费用多久？
A: Vercel 和 Netlify 个人项目永久免费
